Endangered Birds Stamps

Challenge

To create a set of four stamps about an endangered species.

Solution

I decided to focus on endangered birds. I wanted a unifying element that could go throughout all four stamps while also working on each individual stamp. I drew the birds in illustrator because I wanted each bird's unique colors to pop. I used the tree branch as the unifying element.
